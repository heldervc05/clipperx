# 💈 ClipperX - Sua Barbearia a um Clique

> **Projeto Acadêmico Final | Disciplina de Front-end** > Uma plataforma moderna para localização de barbearias, agendamento de horários e marketplace de produtos masculinos.

---

## 💡 O Conceito
O **ClipperX** foi desenvolvido para simplificar a conexão entre clientes e profissionais da beleza masculina. O projeto resolve três dores principais em uma única interface:
1. **Descoberta:** Encontrar barbearias de forma rápida.
2. **Agendamento:** Sistema simples para marcar horários sem complicações.
3. **ClipperX Shop:** Uma vitrine de produtos (pomadas, óleos e acessórios) integrada ao ecossistema da plataforma.

## 🌟 Destaques do Desenvolvimento
Como foco do trabalho final, priorizamos a construção de componentes robustos e fluxos de usuário claros:
* **Telas de Autenticação:** Login e Cadastro exemplares com validações de campo.
* **Página Shop:** Layout de e-commerce clean, focado na exibição de produtos.
* **Interface Responsiva:** Adaptada para que o usuário possa agendar seu corte pelo celular ou desktop.

## 🛠️ Stack Tecnológica
Para garantir performance e segurança, o projeto utiliza:
* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Linguagens:** **TypeScript** (Tipagem estruturada) e **JavaScript** (Lógica funcional).
* **Estilização:** **CSS Customizado** (Layout exclusivo e estilização de componentes).
* **Fontes:** Otimização nativa com Geist e `next/font`.
